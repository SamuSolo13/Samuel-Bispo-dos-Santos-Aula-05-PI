# configuracao.md

## Atividade da aula do dia 24/03

## Para configurar seu ambiente do início nesta fase inicial, você utilizará o Terminal do Windows.

## Preparar as pastas e instalar as ferramentas:

## 1-Criar Pasta: mkdir django_aulas e cd django_aulas.
## 2-Ambiente Virtual: python -m venv venv.
## 3-Ativar: venv\Scripts\activate.
## 4-Nota: Verifique se aparece (venv) no início da linha para confirmar que funcionou.

### Despois...

## Instalar Django: pip install django.
## Criar Projeto: django-admin startproject meu_projeto.
## Abrir Editor: cd meu_projeto e depois code.

### Apos isso ele abrira o VScode automaticamente.
